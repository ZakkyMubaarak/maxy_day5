Tentang Saya

Zakky Mubaarak

Zakky Mubaarak adalah seorang mahasiswa dari Program Studi Sistem Informasi di Universitas Bina Sarana Informatika (UBSI) Ciledug. Dengan latar belakang pendidikan yang kuat dalam sistem informasi, Saya memiliki pengetahuan yang mendalam dalam pengembangan perangkat lunak dan desain sistem. Saat ini, Saya tengah fokus pada pengembangan keterampilan dalam backend development, termasuk pembuatan dan pengelolaan API serta autentikasi menggunakan JWT. Selain itu, Saya juga memiliki pengalaman dalam pengembangan web dengan menggunakan HTML, CSS, dan framework Bulma. Dengan semangat untuk terus belajar dan berkembang, Saya berkomitmen untuk mencapai keahlian yang lebih tinggi di bidang teknologi informasi dan memberikan kontribusi yang signifikan dalam industri ini.
